# 🛸 PGI Drones — Интернет-магазин дронов и комплектующих

Высокотехнологичный, премиальный интернет-магазин профессиональных FPV-дронов, систем РЭБ и обнаружителей БПЛА с локальной базой данных **SQLite** и дублированием заказов/заявок в **Telegram-бота**.

---

## 🛠️ Стек технологий

*   **Бэкенд:** Node.js, Express, SQLite3 (реляционная СУБД в одном файле), dotenv (переменные окружения).
*   **Фронтенд:** Чистый HTML5, CSS3 (переменные, анимации, адаптивная сетка Flexbox/Grid), JavaScript ES6+ (AJAX-запросы к API бэкенда, LocalStorage для корзины и избранного).
*   **Интеграция:** Нативный Node.js HTTPS клиент для безопасной работы с **Telegram Bot API**.

---

## 📂 Структура проекта

*   `/server` — серверное приложение Express, сервис базы данных и отправки уведомлений в Telegram.
*   `/server/db/database.sqlite` — локальный файл базы данных SQLite (создается автоматически при первом запуске).
*   `/server/db/schema.sql` — SQL-схема таблиц и автозаполнение (сиддер) товаров.
*   `/public` — статические файлы фронтенда (HTML, CSS, JS, изображения).
*   `.env` — файл настроек (порт сервера, токены Telegram).

---

## ⚙️ Настройка Telegram-бота

Чтобы заказы и заявки с сайта мгновенно приходили к вам в Telegram:

1.  **Создайте Telegram-бота:**
    *   Найдите бота **@BotFather** в Telegram.
    *   Отправьте команду `/newbot` и следуйте инструкциям.
    *   Скопируйте полученный **Токен API** (например, `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ`).
2.  **Получите ваш Chat ID:**
    *   Найдите бота **@userinfobot** или **@GetChatID_Bot** в Telegram и отправьте ему любое сообщение.
    *   Скопируйте ваш числовой **Chat ID** (например, `987654321`).
3.  **Заполните файл `.env`:**
    *   Откройте файл `.env` в корневой папке проекта.
    *   Вставьте ваши токен и Chat ID:
        ```env
        PORT=3000
        TELEGRAM_BOT_TOKEN=123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ
        TELEGRAM_CHAT_ID=987654321
        ```

*(Если вы не настроите токены, бэкенд продолжит работать в режиме разработчика и будет выводить все уведомления о заказах прямо в консоль сервера, так что вы все равно сможете их протестировать).*

## 🚀 Запуск проекта

1.  Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).
2.  Откройте терминал в папке проекта `pgi-drones`.
3.  Установите зависимости (если еще не установлены):
    ```bash
    npm install
    ```
4.  Запустите сервер в режиме разработки:
    ```bash
    npm run dev
    ```
5.  Откройте в браузере: **[http://localhost:3000](http://localhost:3000)**

## 💡 Функциональные возможности

1.  **Полная автономность:** Сайт работает полностью локально без внешних зависимостей. Товары считываются из SQLite.
2.  **Динамический поиск и фильтры:** В шапке встроен живой поиск. На странице каталога работают фильтры по категориям (Дроны, РЭБ, Комплектующие, Детекторы) и статусам (Новинки, Акции, Популярное) в реальном времени.
3.  **Корзина и Лайки:** Работает добавление товаров в корзину с регулированием количества и в избранное (лайки) с сохранением состояния в `LocalStorage`.
4.  **Безопасный расчет цен:** При оформлении заказа итоговая стоимость рассчитывается на стороне сервера, считывая актуальные цены из СУБД SQLite (защита от изменения цен на клиенте).
5.  **Премиальный дизайн:** Современная темная тема с эффектами матового стекла (backdrop-blur), плавным выезжающим меню, микро-анимациями кнопок и неоновым свечением при наведении.
